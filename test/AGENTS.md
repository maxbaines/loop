# AGENTS.md - Hello World Mac App

## Project Type

This is a **simple macOS application** built with Swift and SwiftUI.

## Requirements

- macOS 13.0+ (Ventura or later)
- Swift 5.9+
- Xcode Command Line Tools

## Build Instructions

```bash
# Build the app
swift build

# Run the app
swift run
```

## Code Standards

- Use SwiftUI for the UI
- Follow Swift naming conventions
- Keep it simple - this is a Hello World app

## What NOT To Do

- Don't overcomplicate the app
- Don't add unnecessary dependencies
- Don't create an Xcode project - use Swift Package Manager
